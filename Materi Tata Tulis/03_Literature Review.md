• Software testing adalah proses untuk menguji perangkat lunak untuk
memastikan bahwa perangkat lunak tersebut berjalan dengan baik.
Menurut Roger Pressman (2014), pengujian perangkat lunak adalah
proses untuk menemukan kesalahan.
• Menurut Ian Sommerville (2016), pengujian perangkat lunak
bertujuan untuk memvalidasi dan memverifikasi perangkat lunak.
• Test case prioritization adalah teknik untuk mengurutkan test case.
Penelitian oleh Smith (2020) menggunakan algoritma genetika.
Penelitian oleh Ahmad (2021) menggunakan greedy algorithm.
Penelitian oleh Lee (2022) menggunakan machine learning.
